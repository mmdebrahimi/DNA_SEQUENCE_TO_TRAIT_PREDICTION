# HCMV antiviral-resistance cell — v0 (2026-07-23)

**The first herpesvirus decoder cell**, and the 4th built against a FREE, MEASURED, INDEPENDENT viral
phenotype (after HIV-1 / SARS-CoV-2 Mpro / influenza NA). Selected from the free-label round-2 research
(`wiki/free_label_acquisition_round2_2026-07-23.md`, top GO candidate: lowest build-risk because it reuses the
exact validated HIV/SARS-CoV-2 architecture).

## What it decodes

`dna-amr --drug <drug> --observed UL97:M460V,UL54:F412L[,UL56:C325W]` (wheel-only, no BLAST/Docker):

| Drug | Target gene(s) | Mechanism |
|---|---|---|
| ganciclovir / valganciclovir | UL97 (activation) + UL54 (pol) | UL97 loss-of-activation OR UL54 pol resistance |
| cidofovir | UL54 | pol resistance |
| foscarnet | UL54 | pol resistance (distinct pol domain) |
| letermovir | UL56 (terminase) | C325W/Y high-grade (>8000×); R369/V236 mid-grade |

**MUTANT-LEVEL** (not position-based), because HCMV genes carry benign polymorphisms — a position rule would
over-call. **Per-mutation cross-resistance is catalogued, not assumed** (e.g. UL54 Q578H = GCV+CDV+FOS; S585A =
FOS-only). A phenotyped-**BENIGN** class (36 polymorphisms) is the specificity anchor (the HCMV analogue of
SARS-CoV-2 Mpro P132H).

## Provenance (no fabrication)

Catalog sourced VERBATIM from the Chou-lab recombinant-phenotyping open-access literature — every entry tied to
a fetched source, unverifiable positions EXCLUDED (not invented):
- **PMC3262590** (primary) — UL54 fold-change Table 1 + the recombinant no-effect BENIGN list (Table 2) + canonical UL97 GCV set.
- **PMC5483911** — UL97 codon 591-603 point + deletion fold-changes.
- **AAC 10.1128/aac.00922-18** + **PMC9759347** — UL56 letermovir EC50 table + compilation + benign class.

~52 resistant catalog entries (UL97 20 / UL54 18 / UL56 ~14) + 36 benign. Numbering: AD169 codon convention.
**Excluded (unverified):** UL97 A594T/L595F/C607F/C607Y; UL56 E237G; maribavir (UL27/UL97) = a separate future cell.

## Honest validation tier

**IN-DISTRIBUTION knowledge baseline** — the catalog is curated FROM the measured fold-changes, so scoring it
against them is not independent (same posture as the SARS-CoV-2 CoV-RDB cell). The trust surface renders
`UNKNOWN` (no standing report card) rather than borrowing a metric. An **INDEPENDENT number (v0.1)** needs
HELD-OUT recombinant-phenotyping studies not used to build the catalog — the free measured phenotype exists
(the whole reason this cell was chosen), so it is a real, reachable next step.

## Deferred (v0.1)

- **Genome-FASTA mode** — committed UL97/UL54/UL56 CDS references + a BLAST codon-mapper (mirrors the HIV /
  SARS-CoV-2 genome paths); v0 is wheel-only observed-mode.
- **Independent validation** — score against held-out phenotyping studies for a non-circular number + a
  standing HCMV report card.
- **In-frame deletions** — UL97 del591-594 etc. are catalogued as opaque keys; the genome caller (v0.1) emits them.
- **Per-drug differential within a UL54 cross-resistance domain**; the maribavir (UL27/UL97) cell.
